##  About The Project

This repository contains a complete collection of **108 Java programming exercises** designed to progressively improve programming skills from beginner to advanced level.

The exercises cover:

*  Java Fundamentals
*  Control Structures
*  Arrays & Strings
*  Methods & Functions
*  Object-Oriented Programming (OOP)
*  Inheritance & Polymorphism
*  Interfaces & Abstract Classes
*  Design Patterns
*  Recursion
*  Algorithms
*  Java Streams API
*  Reflection API
